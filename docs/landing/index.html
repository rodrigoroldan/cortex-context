CLI

cortex-context — your terminal companion

A lightweight CLI to ingest, query and manage your knowledge graph without leaving the terminal.

cortex-context ingest Ingests all active dimensions into Neo4j. Idempotent — safe to re-run anytime.
cortex-context ingest --dim spec Ingests only the spec dimension. Fast incremental updates after writing new specs.
cortex-context query "payments" Full-text cross-dimension search with 1-hop graph expansion. Returns matched nodes + neighbors.
cortex-context doctor Health check — validates config, Neo4j connectivity, node counts and embedding provider.
cortex-context status Graph stats: node count per dimension, last ingest timestamp and API health.

Quick Start

Up in 5 minutes

Three commands to a working knowledge graph.

1Clone & configure
git clone cortex-context cp .env.example .env # Set WORKSPACE_ROOT to your repo
2Start the graph
docker compose up -d # Starts Neo4j + Cortex API
3Ingest & query
cortex-context ingest cortex-context query "payments" # ✅ Your agent has context
